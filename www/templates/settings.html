GAME TOOLS

Randomize Cage Operators

Operator types for cages will be randomly generated.
Note: If this option is not selected the No Negative Results option is not available.

No Negative Results

Simple numeric operations will never return negative results.
Note: Sort in Descending Order is mandatory when this value is set.

No Fractions

Simple numeric division will display Whole result with Remainder.

Sort in Descending Order

Simple numeric values will be sorted in descending order when composing the equation for non commutative operations.
